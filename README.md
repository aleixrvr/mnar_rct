This is the repository for the paper "Estimating Treatment Effects with Missings Not At Random in the Estimand Framework using Causal Inference" by A.Ruiz de Villa and Ll. Badiella.

### Installation

Execute the file `install_packages.R` to install the required packages for the code

### Shiny App

You can run the Shiny app by executing the `missing ate app.R` file. This will launch a local server where you can interactively play around with the parameters and see how they affect the results.


### Simulations

For running the simulations, execute the `code>experiments/execute.R` file. The results will be stored in the `results/` folder. You can add new configurations in the `configs` folder following the naming `mnar_*.R` and `questions_*.R`.
